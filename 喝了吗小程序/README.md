# "喝了吗"饮水提醒小程序

一款帮助用户养成良好饮水习惯的微信小程序，通过游戏化的勋章系统和直观的进度展示，让喝水变得更有趣。

## 功能特点

### 核心功能
- **每日饮水打卡**：快速记录每日饮水量
- **目标设定**：自定义每日饮水目标（6-12杯）
- **智能提醒**：可配置的饮水提醒间隔（45/60/90分钟）
- **免打扰模式**：设置睡眠时段，避免打扰
- **数据统计**：周/月饮水数据可视化图表
- **成就系统**：12种不同的饮水成就勋章
- **个人档案**：个人信息和设置管理

### 技术特点
- **原生小程序开发**：使用微信小程序原生框架
- **本地存储**：使用 `wx.setStorageSync` 存储用户数据
- **Canvas绘图**：实现自定义图表和进度条
- **微信API**：集成微信授权、通知等功能
- **响应式设计**：适配不同尺寸的手机屏幕

## 页面结构

```
├── login/           # 微信登录页面

├── stats/           # 饮水统计页面
├── medals/          # 成就勋章页面
└── settings/        # 提醒设置页面
```

## 开发环境

- 微信开发者工具
- 微信小程序基础库 2.19.4+

## 安装和开发

1. 下载微信开发者工具
2. 打开微信开发者工具
3. 导入项目（选择项目根目录）
4. 配置小程序AppID
5. 点击编译运行

## 项目配置

在 `project.private.config.json` 文件中配置：
- `appid`：你的小程序AppID
- `projectname`：项目名称

## 数据存储

### 用户信息
```javascript
{
  userInfo: {
    nickName: '用户昵称',
    avatarUrl: '用户头像URL'
  }
}
```

### 每日目标和设置
```javascript
{
  dailyGoal: 8,              // 每日目标杯数
  reminderInterval: 60,      // 提醒间隔（分钟）
  quietModeEnabled: true,    // 是否开启免打扰模式
  quietStartTime: '23:00',   // 免打扰开始时间
  quietEndTime: '07:00',     // 免打扰结束时间
  notificationsEnabled: true // 是否开启通知
}
```

### 每日记录
```javascript
{
  todayRecords: [
    { time: '08:30', volume: 1 },
    { time: '10:15', volume: 1 }
  ]
}
```

## 通知系统

## 成就系统

共12种成就勋章，包括：
- 初尝甘露：第一天打卡
- 水滴石穿：连续打卡7天
- 饮水达人：累计喝水100杯
- 早起第一杯：08:00前喝水
- 夜猫子饮水：22:00后喝水
- 完美一天：完成每日目标
- 坚持就是胜利：连续打卡14天
- 健康守护者：连续打卡30天
- 水杯收藏家：累计喝水500杯
- 超级饮水王：单日喝水超过15杯
- 习惯养成：连续打卡90天
- 饮水大师：累计喝水1000杯
